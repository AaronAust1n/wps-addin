使用帮助

关于AI助手

AI助手是一款基于人工智能技术的WPS文字插件，旨在帮助用户提高文档编辑效率。插件集成了多种AI功能，包括文本续写、文本校对、文本润色、文本摘要和全文总结等。

功能使用指南

文本续写

基于已有内容智能续写，帮助您快速完成文档。

  1. 在文档中选中一段文本作为续写的基础
  2. 点击功能区中的"文本续写"按钮
  3. 在任务窗格中设置续写选项（长度、风格等）
  4. 点击"生成续写"按钮
  5. 生成结果后，可以点击"插入文档"将内容插入到当前位置
文本校对

检查并修正文档中的错误，包括拼写、语法和语义错误。

  1. 选中需要校对的文本
  2. 点击功能区中的"文本校对"按钮
  3. 在任务窗格中选择校对选项
  4. 点击"开始校对"按钮
  5. 对于检测到的错误，可以选择应用单个修改或全部应用
文本润色

改进文档的表达和流畅度，使文章更加专业。

  1. 选中需要润色的文本
  2. 点击功能区中的"文本润色"按钮
  3. 在任务窗格中选择语气风格和调整程度
  4. 点击"开始润色"按钮
  5. 生成结果后，可以点击"应用到文档"将润色后的内容替换原文
文本摘要

为选定内容生成简洁摘要，帮助理解核心内容。

  1. 选中需要生成摘要的文本
  2. 点击功能区中的"文本摘要"按钮
  3. 在任务窗格中选择摘要长度和格式
  4. 点击"生成摘要"按钮
  5. 生成结果后，可以选择插入文档或复制摘要
全文总结

分析整个文档并生成总结，适用于较长文档。

  1. 点击功能区中的"全文总结"按钮
  2. 在任务窗格中选择摘要选项（长度、格式、重点）
  3. 点击"生成全文摘要"按钮
  4. 生成结果后，可以选择添加到文档末尾或复制摘要

API设置说明

AI助手需要配置API连接信息才能正常工作。点击功能区中的"API设置"按钮，进入设置页面：

  • API地址：输入您的AI服务API基础URL
  • API密钥：输入访问API所需的认证密钥
  • 模型设置：可为不同功能配置不同的AI模型
  • 高级选项：调整生成文本的最大长度和创造性等参数
